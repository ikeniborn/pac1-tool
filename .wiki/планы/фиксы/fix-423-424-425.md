# FIX-423 / FIX-424 / FIX-425

**Источник:** `docs/superpowers/plans/2026-05-01-fix-423-424-425.md`

## Цель

Исправить три корневые причины провалов из прогона 2026-05-01: баг в contract gate matching (+4 задачи), слишком широкий evaluator bypass для lookup (+2 задачи), отсутствие CRM date anchor gate (+1 задача).

## FIX-423: Contract gate matching bug

**Проблема:** `_match_contract_scope(path, scope)` имеет баг — использует `path.startswith(scope)` вместо `path.startswith(scope + "/")` → scope `/outbox` матчит `/outbox-archive/`.  
**Фикс:** `path.startswith(scope.rstrip("/") + "/")`  
**Файл:** `agent/loop.py` (contract gate)  
**Ожидаемый gain: +4 задачи**

## FIX-424: Evaluator bypass слишком широкий для lookup

**Проблема:** Evaluator bypass применяется ко всем `lookup` задачам (не только "pure read"). Read+write lookup ошибочно bypasses evaluator.  
**Фикс:** Bypass только если `done_ops` пустой (нет write/delete операций).  
**Файл:** `agent/evaluator.py`  
**Ожидаемый gain: +2 задачи**

## FIX-425: CRM date anchor gate

**Проблема:** CRM задачи с датами ("update last_contacted to today") не привязывают дату к vault date → используют системное время.  
**Фикс:** Gate в loop.py: при write в `/contacts/` и наличии `last_*_on` полей → inject vault_date hint.  
**Файл:** `agent/loop.py`  
**Ожидаемый gain: +1 задача**
