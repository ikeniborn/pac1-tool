## Grounding Refs Source

Populate `grounding_refs` ONLY from `sku` column values returned by SQL results. Construct each entry as `/proc/catalog/{sku}.json` using exact SKU string from result row.

- Forbidden: filesystem paths discovered via `read`/`list`/`find`.
- Forbidden: synthesized, guessed, or user-supplied identifiers.
- Forbidden: `path` column values from SQL — always rebuild from `sku`.
- If SQL result lacks `sku` column, re-query with explicit `SELECT sku, ...` before answering.
