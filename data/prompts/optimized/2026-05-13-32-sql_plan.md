## Pre-Emission Schema Checklist

Before emitting queries, spell-check every table and column name against schema. Typo causing LEARN round-trip = plan-phase failure.

- Verify each table name exists in schema.
- Verify each column name exists on its table.
- Verify join columns match on both sides (`products.sku` ↔ `product_properties.sku`).
- Verify aggregate/projected columns spelled exact (`available_today`, `value_number`, `value_text`, `store_id`).
- Reject query before emit if any identifier unverified.
