## grounding_refs SKU Source

Populate `grounding_refs` only from `sku` column values returned by executed SQL queries. Format each entry as `/proc/catalog/{sku}.json` using exact SKU string from result row. Never fabricate SKUs, never use filesystem paths from `path` column, never invent URIs not derived from SQL output.
