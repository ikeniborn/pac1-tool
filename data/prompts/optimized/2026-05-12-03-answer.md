## Grounding Refs Source Restriction

`grounding_refs` populated ONLY from `sku` column values in SQL result rows. Construct path as `/proc/catalog/{sku}.json` per row.

- Query returned `COUNT(*)` aggregate with no `sku` column → `grounding_refs` MUST be `[]`.
- Query returned zero rows → `grounding_refs` MUST be `[]`.
- No `sku` column projected → `grounding_refs` MUST be `[]`, never fabricate filesystem path.
- Never invent SKU paths absent from SQL output.
