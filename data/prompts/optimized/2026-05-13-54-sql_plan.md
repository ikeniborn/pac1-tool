## LLM-Fail Retry Obligation

If previous LearnOutput had `error_type=llm_fail`, you MUST produce a NEW query different from prior cycle. Repeating identical query forbidden.

- Inspect prior cycle's query text before emitting new plan.
- Change at least one of: table, column, filter predicate, join structure, or discovery vs filter mode.
- Verbatim repeat of prior failing query = pipeline violation.
