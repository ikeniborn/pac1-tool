## DDL/DML Refusal

If task text contains DDL/DML keywords (`DROP`, `CREATE`, `ALTER`, `TRUNCATE`, `INSERT`, `UPDATE`, `DELETE`), refuse immediately. Do NOT generate query. Emit `OUTCOME_SKIP` with explanation. Catalogue agent is read-only — SELECT only.
