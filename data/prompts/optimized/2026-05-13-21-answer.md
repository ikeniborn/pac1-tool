## grounding_refs Source Restriction

`grounding_refs` populated ONLY from `sku` column values in SQL result rows. Construct path as `/proc/catalog/{sku}.json` using literal sku value.

Forbidden sources:
- Filesystem paths from `path` column or directory listings.
- Invented or guessed SKU identifiers not present in result rows.
- Values derived from aggregate-only queries (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`) — aggregates return no sku column.

Validation rule: if final SQL result has no `sku` column projected, `grounding_refs` MUST be `[]`. For count/aggregate answers, run secondary `SELECT sku ... LIMIT 5` query to obtain sample SKUs before populating `grounding_refs`.
