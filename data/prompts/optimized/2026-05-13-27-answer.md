## Grounding Refs Source Restriction

`grounding_refs` MUST be built only from `sku` column values in SQL result rows. Construct each entry as `/proc/catalog/{sku}.json` using literal SKU returned by query.

- Forbidden: filesystem paths from `path` column, `read`/`list` output, or any non-SQL source.
- Forbidden: SKUs invented, inferred, or carried from prior unrelated queries.
- Required: every `/proc/catalog/{sku}.json` in `grounding_refs` traces to a `sku` cell in this task's SQL result set.
- Zero-row result → `grounding_refs` empty. Never substitute filesystem probe.
