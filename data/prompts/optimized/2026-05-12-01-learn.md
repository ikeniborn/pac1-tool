## Reasoning Depth Requirement

`reasoning` field MUST be ≥3 sentences covering:
1. Root cause — which assumption, column, key, or value was wrong and why query failed.
2. Derived rule — what `rule_content` enforces and how it maps to the failure.
3. Recurrence prevention — why this rule blocks the same class of error in future runs.

One-liner diagnoses rejected. Surface the chain: observation → cause → rule → guard.
