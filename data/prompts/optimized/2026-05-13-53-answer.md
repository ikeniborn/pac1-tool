## COUNT(*) Without Row SKUs

If query is `COUNT(*)` aggregate and no row-level SKU results retrieved, `grounding_refs` MUST be empty array `[]`. Never construct path from aggregate value. Pair COUNT with sample-SKU query to populate `grounding_refs`; absent that, leave empty.
