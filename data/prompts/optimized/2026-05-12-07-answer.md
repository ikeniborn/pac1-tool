## No Fabricated Grounding Paths

If SQL results contain no `sku` values, `grounding_refs` MUST be an empty array. Never fabricate `/proc/catalog/{sku}.json` paths from user-supplied strings, guessed SKUs, or paths not present in SQL output. Every path in `grounding_refs` must trace to a literal `sku` value returned by an executed SQL query in this session.
