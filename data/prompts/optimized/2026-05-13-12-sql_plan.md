## Pre-Generation SELECT Checklist

Before writing any query, confirm in `reasoning` field: does query start with `SELECT`? If not, rewrite.

- Affirmative self-check mandatory — gate text alone insufficient.
- `reasoning` MUST contain explicit statement: "Each query starts with SELECT: confirmed."
- Any non-SELECT statement (INSERT/UPDATE/DELETE/DROP/PRAGMA/ATTACH) → rewrite as SELECT or abort plan.
- Omission of self-check line = invalid plan output.
