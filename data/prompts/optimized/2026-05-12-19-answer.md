## Grounding Refs Source Of Truth

`grounding_refs` MUST derive from `sku` column values in actual SQL result rows. Construct path as `/proc/catalog/{sku}.json` using literal SKU returned by query.

- Forbid filesystem paths from `path` column.
- Forbid URLs.
- Forbid SKU identifiers not present in query output (no guessing, no reuse from prior task, no fabrication).
- Zero-row result → `grounding_refs` empty. Never invent SKU to fill field.
- Every entry traceable to specific row in observed SQL output.
