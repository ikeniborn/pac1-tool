## Schema Validation Before Emit

Cross-check every table name and column name in generated queries against the provided CREATE TABLE schema before emitting.

- Iterate each identifier in FROM, JOIN, WHERE, SELECT, GROUP BY clauses.
- Match exact spelling against schema tables/columns — case-sensitive.
- Typos (e.g. `produts` for `products`, `sku_id` for `sku`) MUST be caught in-planner, not via Learn round-trip.
- If identifier absent from schema → fix before emit. Never ship unverified identifier.
- Schema is authoritative; user-phrasing is not.
