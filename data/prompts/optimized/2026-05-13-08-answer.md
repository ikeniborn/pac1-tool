## COUNT Aggregate Grounding

When final SQL is `COUNT(*)` with no `GROUP BY` and result rows contain no `sku` column, `grounding_refs` MUST be empty array `[]`. Populate `grounding_refs` only when result rows expose `sku` values. Aggregate-only result without per-row SKUs cannot be grounded to specific catalog files.
