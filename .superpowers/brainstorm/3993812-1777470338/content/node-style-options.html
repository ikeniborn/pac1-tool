Как совместить цвет по типу, размер по uses, и уровень уверенности (confidence)?

A — Цвет по типу, размер по uses

■ insight  ■ rule  ■ antipattern  ■ pattern
Размер = base + log(uses) × scale. Confidence — через прозрачность (opacity = 0.4 + confidence × 0.6)

B — Заливка по confidence, рамка по типу

Цвет заливки: зелёный=высокий → жёлтый=средний → красный=низкий. Рамка/форма = тип. Размер = uses.

C — Цвет по типу + яркость по confidence + размер по uses

Все три измерения на одном узле: цвет (тип) + opacity (confidence) + radius (uses). Максимум информации, минимум визуального шума.

Моя рекомендация — вариант C: три измерения совмещены, легенда объясняет всё. Выбери или скажи в терминале что изменить.
