Дизайн: scripts/visualize_graph.py

FastAPI + vis.js · один файл · uv run python scripts/visualize_graph.py

Архитектура

scripts/visualize_graph.py
├─ FastAPI app
GET / → HTML-страница с vis.js
GET /api/graph → JSON {nodes, edges}
└─ uvicorn.run(host=0.0.0.0, port=8765)
Запуск:
uv run python scripts/visualize_graph.py
→ открывает http://localhost:8765

API: GET /api/graph

Параметры (опционально):
?tag=email — фильтр по тегу
?type=rule — фильтр по типу узла
?min_confidence=0.7 — порог уверенности
Ответ: {nodes: [...], edges: [...]}
Узел: {id, label, title(tooltip), color, size, opacity}

Визуальное кодирование

Цвет по типу
● insight — синий
● rule — зелёный
● antipattern — красный
● pattern — оранжевый
Размер и яркость
Радиус = 10 + log(uses+1) × 8
Opacity = 0.4 + confidence × 0.6
Hover-tooltip: type, tags, text,
confidence, uses, last_seen

UI-панель управления

🎛️ Чекбоксы типов — insight / rule / antipattern / pattern
🔍 Поиск по тексту — фильтрует узлы на лету
📊 Слайдер confidence — скрывает узлы ниже порога
🔄 Кнопки layout — forceAtlas2 / hierarchical / circular
📌 Drag узлов — vis.js из коробки
🔎 Zoom/pan — колесо мыши / drag фона
Зависимости: добавить fastapi и uvicorn в pyproject.toml (optional group viz). Запуск: uv run python scripts/visualize_graph.py
